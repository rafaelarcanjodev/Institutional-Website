---

## 🎯 Academic & Professional Synergy

This project serves as the **Capstone Integrator** for the Frontend Development module. It represents the intersection of rigorous academic standards at **Católica SC** and real-world market demands. The mission was to deliver a professional-grade institutional website for a local community partner, applying advanced UI/UX principles.

## ✨ Technical Deliverables

* **Advanced Responsiveness:** Implemented a mobile-first architecture using Bootstrap’s grid system and custom CSS media queries.
* **UX Strategy:** Translated stakeholder requirements into a functional, user-centric interface, focusing on accessibility and speed.
* **Dynamic Interactivity:** Utilized JavaScript and jQuery for DOM manipulation and to enhance the end-user experience.
* **Industry Standards:** Developed following clean code practices and performance optimization techniques taught throughout the semester.

---

## 🛠️ Tech Stack

* **Structure:** HTML5.
* **Styling:** CSS3 & Bootstrap 5.
* **Logic:** JavaScript / jQuery.
